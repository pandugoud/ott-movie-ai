## 📌 Features

🎥 Netflix-style UI  
🤖 AI Movie Recommendation System (TF-IDF based)  
🔎 Search & Explore Movies  
❤️ Watchlist System  
📊 Analytics Dashboard  
🎬 Movie Trailer Support (YouTube API ready)  
💬 AI Assistant (Rule-based / GPT upgradeable)  
☁️ Ready for Streamlit Cloud Deployment

## 🧠 Tech Stack

- Python 🐍  
- Streamlit 🌐  
- Pandas 📊  
- Scikit-learn 🤖  
- OpenAI API (optional upgrade)  
- YouTube Data API (optional)
